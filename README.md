# huber-vs-repeated-median
Tema ovog projekta je poređenje algoritama `Huber` i `Repeated Median`, uz `Least Squares` kao osnovno, najjednostavnije rešenje koje služi za referencu. Svi algoritmi se pokreću nad sirovim podacima i nad podacima sa dodatim ekstremnim vrednostima, kako bi se uočila razlika u tome kako reaguju na njih.

Obradjivani skup podataka je ['Fish market'.](https://huggingface.co/datasets/scikit-learn/Fish/resolve/main/Fish.csv)
